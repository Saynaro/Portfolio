## ✨ Key Features

- **🌊 Smooth Scrolling**: Integrated with [Lenis](https://github.com/darkroomengineering/lenis) for a buttery-smooth user experience.
- **🎨 Modern Animations**: Powered by [GSAP](https://greensock.com/gsap/) for high-performance motion and entrance effects.
- **🍱 Responsive Design**: Fully optimized for mobile, tablet, and desktop using Tailwind CSS 4.
- **🖱️ Custom Animated Cursor**: A unique, interactive cursor that reacts to user interactions.
- **📂 Project Showcase**: Detailed sections for my latest works with direct links and descriptions.
- **🛠️ Tech Stack Visualization**: A clean, organized display of both Frontend and Backend skills.

## 🛠️ Tech Stack

### Frontend
- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4 (Next-gen CSS utility engine)
- **Animation**: GSAP (GreenSock Animation Platform) + `@gsap/react`
- **Scrolling**: Lenis (Smooth scroll library)
- **Icons**: Lucide React & React Icons

### Backend (Core Competencies)
- **Runtime**: Node.js & Express
- **ORMs**: Prisma
- **Databases**: MySQL, MariaDB, PostgreSQL
- **Security**: JWT (JSON Web Tokens) & API Security best practices

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS version recommended)
- `npm` or `yarn`

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Saynaro/Portfolio.git
    cd Portfolio
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```

## 📦 Deployment

This project is configured for automated deployment to **GitHub Pages** using the `gh-pages` package.

To deploy:
```bash
npm run deploy
```

Or use the "shipit" shortcut to commit, push, and deploy in one command:
```bash
npm run shipit
```
